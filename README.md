cours-du-soir-bdd
=================

Behaviour Driven Development par @lepinay

Présentation des motivations qui se cachent derrière BDD, bonnes pratiques/choses à éviter.
Cette session aura pour focus principal la méthodologie en elle même, BDD étant avant tout un processus plus qu'un "outillage technique".
Tous les profils sont donc les bienvenus (Dev,Testeurs,Analystes métier ou PO).
